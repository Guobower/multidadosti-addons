Partner Is Not Customer

This module customize ResPartner Module

This Module set the value of 'customer' field from 'res.parter' model as 'False' when is a contact. In the 'res.partner' model, the original value of 'customer' field is True. This module, change this default value of this field as 'False'. That way, the new partner added will not be set customers by default when he is a contact.
